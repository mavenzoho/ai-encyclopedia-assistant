## Recording Tips

1. **Do a dry run first** - make sure the app generates successfully before recording
2. **Pre-warm the API** - generate one page before recording so subsequent calls are faster
3. **Use Chrome** - required for AudioWorklet API
4. **Close notifications** - prevent popups during recording
5. **Record audio separately** if your screen recorder audio quality is poor - sync in editing
6. **Keep mouse movements smooth** and deliberate
7. **Pause briefly** after each section before transitioning
8. **If the API is slow**, narrate what's happening ("Encyclopia is generating illustrations right now...")
9. **Show the GCP console briefly** or record a separate short clip for the proof of deployment requirement
10. **Keep it under 4 minutes** - only the first 4 minutes are evaluated

## Proof of Google Cloud Deployment

Record a **separate short clip** (30-60 seconds) showing:
- Google Cloud Console open
- Cloud Run services page showing your `ai-encyclopedia` service
- The service URL, region, and status
- Optionally: Cloud Build history showing the build
- OR: Link to `deploy.sh` in your repo as proof of GCP usage
